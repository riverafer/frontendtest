# frontendtest

## Get started

### Clone the repo

```shell
git clone https://github.com/riverafer/frontendtest
cd frontendtest
```

### Install npm packages

Install the `npm` packages described in the `package.json` and verify that it works:

```shell
npm install

ng serve -o to open 4200 server angular default
```
